# SmartyGui
SmartyGUI - mod that adds a bunch of GUI things to your minecraft!

You can easily configure it to disable some modules in this mod.

Currently adds:

    Main Menu additional Info
    Show Version
    Show Block
    ArmorStatusHUD
    Tooltip
    Furnace Info
    Player List (if server side installed too!)
